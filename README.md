> Startup | Clone Repo
```
git clone https://github.com/disit/snap4city-docker
```
> Entry in Dir
```
cd snap4city-docker/DataCity-Small
```

> Permissions Dirs/Files & MAX Count
```
sudo ./setup.sh
```

> Pulling Docker Containers 
```
docker-compose up -d
```

> Launch Docker Services
```
sudo ./post-setup.sh
```
