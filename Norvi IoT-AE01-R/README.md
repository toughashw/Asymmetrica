> DIGITAL INPUT/OUTPUT 
- I0 –-> GPIO18
- I1 –-> GPIO39  
- I2 –-> GPIO34  
- I3 –-> GPIO35  
- I4 –-> GPIO19  
- I5 –-> GPIO21  
- I6 –-> GPIO22  
- I7 –-> GPIO23 

> RELAY OUTPUTS
- R0 –-> GPIO14  
- R1 –-> GPIO12  
- R2 –-> GPIO13  
- R3 –-> GPIO15 
- R4 –-> GPIO2  
- R5 –-> GPIO33

> RS485
- Flow PIN --> GPIO4
- TX --> GPIO5
- RX --> GPIO3
